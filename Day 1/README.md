# **Backend Development 101: Day 1**

## **Terminal Commands**

### **List Current Directory**
```
ls
```

### **Current Directory**
```
pwd
```
### **Change Directory**
```
# Absolute Path
cd C:\User\PC\Desktop

# Relative Path
cd Documents/folder1
```
### **Create Directory**
```
mkdir <directory name>
```

## **Basic Node Syntax**

### *console.log*
- Used to print value to the terminal.
```
console.log(value);
```
